1. Dashboard 📊

El Dashboard es la pantalla principal que ofrece una visión general del estado del negocio. Se actualiza automáticamente y muestra:

  • Resumen Financiero: Tarjetas con el total facturado, saldo pendiente y total recaudado.
  • Gráficos: Comparativas de facturación vs recaudo, distribución de estados de facturas y top de clientes.
  • Alertas: Facturas próximas a vencer y productos con stock bajo.

2. Gestión de Clientes 👥

Módulo para administrar la base de datos de clientes.

Cómo añadir un nuevo cliente:

  1. En el menú lateral, seleccione la opción Clientes.
  2. Haga clic en el botón + Nuevo Cliente ubicado en la parte superior derecha.
  3. Complete el formulario con los datos del cliente:
    • Nombre: Nombre completo o razón social (Obligatorio).
    • NIT / Cédula: Documento de identidad (Obligatorio, sin duplicados).
    • Teléfono, Dirección, Email: Datos de contacto.
  4. Haga clic en Guardar para registrar al cliente.

Buscar y Editar

Utilice la barra de búsqueda para encontrar clientes por nombre o documento. Haga doble clic en un cliente o use el botón de Editar para modificar su información.

3. Facturación 📄

Permite crear, enviar y gestionar facturas de venta.

Cómo hacer una factura:

  1. Vaya al menú Facturas y haga clic en + Nueva Factura (o use el botón de acceso rápido en el encabezado).
  2. En el formulario de nueva factura:
    • Cliente: Busque y seleccione el cliente.
    • Método de Pago: Elija la forma de pago (Efectivo, Transferencia, Crédito, etc.).
    • Fecha: Por defecto es la fecha actual.
  3. Agregar Productos:
    • Busque el producto por nombre o código en la lista desplegable.
    • Ingrese la Cantidad.
    • Verifique el Precio (puede ajustarlo si es necesario).
    • Haga clic en + Agregar. Repita para todos los productos deseados.
  4. Verifique los totales (Subtotal y Total) en el resumen derecho.
  5. Haga clic en ✓ Crear y Enviar para finalizar y guardar la factura.

Cómo imprimir una factura:

  • Desde el listado: En el módulo Facturas, busque la factura deseada en la tabla y haga clic en el icono de impresora (🖨) en la columna "Acciones".
  • Desde el borrador: Mientras crea una factura nueva, puede hacer clic en 🖨 Imprimir para ver una vista previa o imprimir un borrador antes de guardarla.

Nota: Una vez creada la factura, el stock de los productos seleccionados se descuenta automáticamente del inventario.

4. Pagos y Recaudos 💰

Registro de abonos y pagos completos a facturas.

Cómo hacer un pago:

  1. Ingrese al módulo Pagos desde el menú lateral.
  2. Haga clic en + Nuevo Pago.
  3. Complete la información del pago:
    • Factura: Seleccione la factura pendiente de pago de la lista desplegable.
    • Monto: El sistema sugerirá el saldo pendiente. Puede modificarlo para hacer un abono parcial.
    • Método de Pago: Seleccione cómo se recibe el dinero.
  4. Verifique el resumen y haga clic en ✓ Registrar Pago.

Anulación de Pagos:

Si cometió un error, busque el pago en la lista histórica ("Historial de Pagos") y haga clic en el botón Anular (papelera/icono rojo). El saldo se devolverá a la factura correspondiente.

5. Cartera 📉

Herramienta para el seguimiento de cuentas por cobrar.

  • Visualice todas las facturas pendientes agrupadas por cliente.
  • Consulte la Edad de Saldos para identificar deudas vencidas (30, 60, 90 días).
  • Identifique clientes con saldo a favor para aplicar en futuras compras.

6. Balance Financiero 📈

Reportes detallados del rendimiento del negocio.

Puede filtrar por períodos (Diario, Mensual, Anual) para ver:

  • Total facturado vs Total costos.
  • Utilidad bruta estimada.
  • Flujo de caja por método de pago (Efectivo, Transferencia, Crédito).

7. Inventario 📦

Control total de productos y existencias.

  • Productos: Cree y edite productos definiendo precios y stock mínimo.
  • Kardex: Vea el historial completo de movimientos (Entradas y Salidas) de cada producto.
  • Ajustes: Realice ajustes manuales de inventario (por pérdidas, obsequios, etc.) usando la opción "Movimiento".

8. Configuración ⚙

Gestor de Base de Datos

Para realizar tareas avanzadas, vaya a Configuración > Gestor de Base de Datos e ingrese el PIN de seguridad (Por defecto: 0000).

1. Exportar la base de datos (Backup):

Utilice esta opción para guardar una copia de seguridad de toda su información.

  1. En el Gestor, haga clic en la tarjeta Exportar Base de Datos.
  2. Se abrirá una ventana para guardar el archivo. Elija una ubicación segura (ej. Documentos, USB).
  3. Haga clic en Guardar. Se generará un archivo .db.

2. Importar la base de datos (Restaurar):

Permite recuperar información desde una copia de seguridad previamente exportada.

¡Advertencia! Esta acción reemplazará todos los datos actuales.
  1. Haga clic en Importar Base de Datos.
  2. Busque y seleccione el archivo .db de respaldo.
  3. Confirme la acción.
  4. La aplicación se reiniciará automáticamente para aplicar los cambios.

3. Inicializar el programa (Limpiar Tablas):

Si desea reiniciar las operaciones (borrar facturas y movimientos de prueba) pero mantener sus clientes y productos, siga estos pasos:

  1. En el Gestor, seleccione la opción Limpiar Tablas Específicas.
  2. Marque las casillas de las tablas operativas:
    • Factura (Elimina todas las facturas).
    • ItemFactura (Elimina los detalles de facturas).
    • Pago (Elimina el historial de pagos).
    • MovimientoInventarioExt (Reinicia el historial de movimientos de stock).
  3. No marque Cliente ni ProductoExt si desea conservar su catálogo.
  4. Haga clic en Limpiar Seleccionadas y confirme las advertencias de seguridad.
